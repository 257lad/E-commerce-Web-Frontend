# MVP Requirements: "VibeShop" (Vue 3)

## Tech Stack
- Frontend: Vue 3 + Vite + TypeScript
- State Management: Pinia
- UI: shadcn-vue + Tailwind

## Core Features
1. **Product Gallery**: Grid using `v-for` to list products.
2. **Stateful Cart**: A Pinia store to handle adding/removing items.
3. **Cart Sidebar**: Using the `Sheet` component from shadcn-vue.
